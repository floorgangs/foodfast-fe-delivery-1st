# Assets

Thư mục này chứa các file assets cho app:

- **icon.png** (1024x1024px): App icon
- **splash.png** (1242x2436px): Splash screen
- **adaptive-icon.png** (1024x1024px): Android adaptive icon
- **favicon.png** (48x48px): Web favicon

## Tạo assets nhanh

Vì đây là mock app, bạn có thể:

1. Tạo các file PNG đơn giản với màu #EA5034
2. Hoặc download từ: https://via.placeholder.com/

```bash
# Download placeholder images (Windows)
curl -o icon.png https://via.placeholder.com/1024/EA5034/FFFFFF?text=FoodFast
curl -o splash.png https://via.placeholder.com/1242x2436/EA5034/FFFFFF?text=FoodFast
curl -o adaptive-icon.png https://via.placeholder.com/1024/EA5034/FFFFFF?text=F
curl -o favicon.png https://via.placeholder.com/48/EA5034/FFFFFF?text=F
```

Hoặc tắt yêu cầu assets trong app.json tạm thời.
